Do the following to use this code...
Create folder Composite Face Code

Put the folder in your Desktop

Put the following files in the Composite Face Code folder
1) Tester/Trainer m-files
2) Main m-file
3) PreprocessDataSetPlus m-file
4) The folders contained in the database folder (POFA, SampleDataSet)
5) Put all the files from the function folder into the Composite Face Code folder.

Use the Main m-file to set paths, run tests, and choose activations/number of layers.
